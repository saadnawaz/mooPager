Using as a Slide show

The script can be easily used as a slide show, just pass 1 as parameter to the elemPerPage attribute

How to use

  • Since, mooPager is a pagination script based on mootools, therefore, first you need to include mootools. You can download latest version from here
  • Download the mooPager from hereand include it in the script
  • If you want to use default classes for the active page numbers and deactive page numbers, then copy and paste the following classes in your html css .active{ color: 'white' } .deActive{ color: 'gray' }
  • Assign all the elements same class name on which you want to apply pagination
  • Instantiate the pagination class onload using;
    new Pager({elemPerPage: 5, className: '.simpleBox'});
  • Options are:
    • elemPerPage: Number of elements you want to display per page. Default to 5
    • className: Name of the class which is applied on all elements
    • effectDuration: Duration of the effect. Possible values are 'short', 'normal', 'long' or time in milliseconds. Default to 'long'
    • noEffect: Whether to include effect or not. Possible values are false and true. Default to true
    • pageNumberHighlightClass: CSS class name to be applied on the highlighted page numbers shown on top of the elements.
    • pageNumberUnHighlightClass: CSS class name to be applied on the unhighlighted page numbers shown on top of the elements.
